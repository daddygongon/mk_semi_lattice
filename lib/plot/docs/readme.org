* plot
** 通常
学習の進捗状況を確認するために
: hc plot
があります．

[[file:./plot/plot.002.png]]

通常のplotでは実行directoryにある
1. check_log.yaml
1. score_log.yaml
を参照して
1. cumulative time(without option)
1. score (-s)
1. word number (-w)
を表示します．

** dirごと，全体
記憶を確実に進めるためには，
: divide and conquer(各個撃破）
が基本．したがって，細かく分けたdirごとに
仕上げていくのがいいです．
また，それらをまとめる(chunking)単語を
directory名に置いておくのがコツ．

しかし，全体の進捗が知りたい時があります．
その時には
: hc -L 2
を使って，cwdと一つ下の合計を表示することができます．
